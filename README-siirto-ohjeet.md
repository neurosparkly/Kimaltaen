# Näin viet tiedostot GitHubiin

1. Lataa zip-paketti.
2. Pura se.
3. Lisää tiedostot repoosi.
4. Siirrä vanhojen sivujen sisältö uusiin tiedostoihin yksi sivu kerrallaan.
5. Päivitä linkit vasta lopuksi.

Turvallinen eteneminen:
- lisää uudet tiedostot ensin
- kopioi sisältö niiden sisään
- testaa sivu
- poista vanhat tiedostot vasta sen jälkeen
